# PuffLess

A SwiftUI iOS app to help you gradually reduce and quit vaping by tracking your usage, setting reduction goals, and monitoring your progress over time.

## Features (Planned)

- Track daily vape usage (puffs, sessions, nicotine level)
- Set gradual reduction goals
- View progress with charts and streaks
- Receive motivational reminders
- Track money saved and health milestones

## Requirements

- iOS 17.0+
- Xcode 16+
- Swift 5.9+

## Getting Started

1. Clone the repo
2. Open `PuffLess.xcodeproj` in Xcode
3. Build and run on simulator or device
